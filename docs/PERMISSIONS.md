# 权限说明

| 能力 | Viewer | Operator | Admin |
| --- | --- | --- | --- |
| 授权直播间看板/分析 | 是 | 是 | 全部 |
| 预警查看 | 授权范围 | 授权范围 | 全部 |
| 预警确认/重推 | 否 | 是 | 是 |
| 导出 | 默认否，可单房间授权 | 授权范围 | 全部 |
| 数据源、同步、指标、班次 | 否 | 否 | 是 |
| 用户权限、密钥、审计 | 否 | 否 | 是 |

权限由 FastAPI 依赖从签名会话和数据库解析。房间筛选项、事实、采集点、详情、导出和预警均拼入服务端允许房间集合；前端隐藏仅用于体验，不作为安全边界。开发模式可用 `DEV_AUTH_BYPASS=true`，生产配置会拒绝该值。
